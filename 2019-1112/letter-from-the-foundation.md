# 基金会来信

亲爱的读者：

我们向你送上最美好的祝福，愿你新的一年幸福快乐。

2019 年接近尾声，我们想借此机会祝贺 FreeBSD 期刊的艺术总监 Dianne Kischitz 即将退休。Journal 自 2014 年创刊以来，封面和内页美术、文章和栏目的排版都出自她手。我们会怀念与她共事的日子，祝她开启人生的下一篇章一切顺利。
